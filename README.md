# Climbing Community

클라이밍 활동을 기록하고, 사용자 간 경험과 정보를 공유할 수 있도록 설계된 **클라이밍 특화 커뮤니티 플랫폼**입니다.

![Uploading climbcommunity.png…]()

---

## 1. 프로젝트 개요

**Climbing Community**는 클라이밍을 즐기는 사용자들이 자신의 활동을 기록하고, 사진·태그·게시글을 통해 서로 소통할 수 있는 SNS형 커뮤니티 서비스입니다.

단순 게시판이 아닌, **운동 기록 + 소셜 기능**을 결합한 구조로 설계하여 지속적인 참여와 커뮤니티 형성을 목표로 했습니다.

* 클라이밍 경험을 축적할 수 있는 기록 중심 서비스
* 관심사 기반 소통이 가능한 커뮤니티 구조

---

## 2. 주요 기능

### 2.1 게시글 작성

* 텍스트 기반 게시글 작성
* 이미지 업로드 지원
* 해시태그(Tag) 기반 분류

### 2.2 커뮤니티 피드

* 최신순 / 태그 기반 게시글 조회
* 사용자 활동 중심 피드 구성

### 2.3 상호작용 기능

* 게시글 좋아요
* 댓글 작성
* 사용자 간 간접적 소셜 관계 형성

### 2.4 사용자 관리

* 회원가입 / 로그인
* 사용자별 게시글 관리

---

## 4. 서비스 흐름

```
회원가입 / 로그인
      ↓
피드 진입
      ↓
게시글 조회
      ↓
게시글 작성 (텍스트 + 이미지 + 태그)
      ↓
커뮤니티 공유
```

사용자는 복잡한 학습 없이 **SNS와 유사한 흐름**으로 서비스를 이용할 수 있도록 설계했습니다.

---

## 5. UI 구성

* **Home Page (피드)**
  게시글 카드 리스트, 이미지 미리보기

* **Write Page**
  텍스트 입력, 이미지 업로드, 태그 선택

* **Post Detail**
  게시글 상세 내용, 댓글 영역

* **Profile**
  사용자 정보 및 작성 게시글 목록

---

## 6. 설계 포인트

* 클라이밍이라는 **명확한 도메인**에 집중한 커뮤니티 설계
* 태그 기반 구조로 게시글 탐색 효율 향상
* 이미지 중심 콘텐츠를 고려한 카드형 UI
* 모바일 사용 환경에 최적화된 인터랙션 설계
